# DevLMS – Learning Management System

DevLMS is a front-end based Learning Management System developed as part of an academic project for the Front-End Technologies subject. The platform is designed to provide an interactive learning experience with structured courses, coding practice, and performance tracking.

## Features

* Course modules for HTML, CSS, JavaScript, and React
* User dashboard with progress tracking
* Live code editor with real-time preview
* Quiz system with score evaluation
* User profile management
* Responsive and clean UI design

## Tech Stack

* HTML
* CSS
* JavaScript
* Bootstrap
* Tailwind CSS

## Screens Included

* Home page (landing UI)
* Dashboard (progress and analytics)
* Course listing page
* Code editor (practice environment)
* Quiz system
* User profile page

## Purpose

The goal of this project is to simulate a real-world learning platform and strengthen front-end development skills, including UI design, component structuring, and user interaction.

## Future Improvements

* Backend integration using Node.js and Express
* Database integration with MongoDB
* Authentication system
* Real-time data persistence
* Deployment on cloud platform

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in your browser
